#📚 Book Store App

## Overview

This is a React Crud application for managing books.
Users can:
- Add books
- View books
- Edit books
- Delete books

The project uses React Hooks, Context API, Custom Hooks, React Router, and JSON Server.

---

# 🚀 Features

- Full CRUD operations
- Global state management using Context API
- Custom Hook for API logic
- Routing with React Router
- JSON Server backend
- Form handling with controlled components
- useRef and useId implementation
